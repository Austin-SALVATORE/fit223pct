# Design Language Pack

## Product Personality

Calm. Premium. Confident. Encouraging. Minimal.

Never feel like military coaching.

## Visual Style

Inspired by: - Apple Fitness+ - Linear - WHOOP - Nike editorial

Avoid: - Neon - Dashboard overload - Bodybuilding clichés

## Typography

Large confident headings. Generous whitespace. Readable workout cards.

## Motion

Every animation must improve understanding.

Examples: - Set completion - Rest timer - Workout transitions - Progress
updates

Avoid decorative motion.

Respect reduced-motion preferences.

## UX Rules

The user should always know: 1. What to do. 2. Why. 3. What comes next.

Minimize typing. Large touch targets. Workout-first navigation.

## Color Philosophy

Dark warm background. High contrast text. Restrained accent colors.
Success should feel subtle rather than flashy.
