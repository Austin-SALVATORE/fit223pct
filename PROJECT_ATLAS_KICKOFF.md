# Product Vision

This is **not** another workout tracker.

It should feel like a combination of:

-   Apple Fitness+
-   Nike Training Club
-   WHOOP
-   Apple Health
-   Strong
-   Hevy
-   Notion
-   A beautifully crafted training journal

The application should be calm, premium, motivating, educational and
deeply personal.

Every design decision should answer:

> Does this make tomorrow's workout easier, more enjoyable or more
> understandable?

If the answer is no, it probably does not belong.

# Product Philosophy

-   Reduce thinking.
-   Increase consistency.
-   Teach instead of overwhelming.
-   Celebrate discipline instead of perfection.
-   Recovery is progress.
-   Information appears only when needed.
-   Motion always has purpose.
-   Never punish missed workouts.
-   Build trust.

------------------------------------------------------------------------

# Training Philosophy

Research evidence-based resistance training before finalizing the
program.

Prefer:

-   ACSM
-   WHO
-   peer-reviewed literature
-   reputable sports science

Avoid influencer-based programming.

Principles:

-   Progressive overload
-   Major muscle groups roughly twice weekly
-   Adapt to available equipment
-   Explain WHY, not only WHAT
-   Use tempo, unilateral work, ROM and volume when load is limited.

------------------------------------------------------------------------

# UX Principles

The user should never wonder:

"What do I do today?"

The application should already know.

Today's workout is always the primary focus.

Weekly reviews appear weekly.

Monthly insights appear monthly.

Avoid dashboard overload.

------------------------------------------------------------------------

# Workout Experience

Workout mode is the heart of the application.

Include:

-   current exercise
-   previous performance
-   editable load
-   reps
-   sets
-   RIR
-   timer
-   progress
-   technique cues
-   substitutions
-   animated exercise explanation
-   satisfying completion interactions

Never interrupt workout flow.

------------------------------------------------------------------------

# Education

Teach progressively.

Examples:

Bench day: Scapular positioning

Romanian Deadlift: Hip hinge

Squat: Bracing

One concept at the right moment.

------------------------------------------------------------------------

# AI Philosophy

No chatbot.

No gimmicks.

AI should quietly:

-   detect stagnation
-   suggest progression
-   recommend substitutions
-   summarize weeks
-   identify trends

Invisible intelligence.
